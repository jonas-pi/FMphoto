# FMphoto

第三方飞牛相册相关的原生鸿蒙应用**预编译包**说明仓库（根目录 `.hap` 为未签名包，仅供熟悉环境的用户自行安装测试）。**源码与完整说明**见：[FNOSforohos](https://github.com/jonas-pi/FNOSforohos)。

## 功能概览

- **登录与首页**：NAS 地址与账号登录（HTTP/HTTPS），首页内嵌飞牛网页并可刷新  
- **图库**：时间线浏览、缩略图与大图预览；支持 **华为 HDR Vivid** 等 HDR 素材在支持机型上的呈现（与素材及「查看原图」等相关）  
- **视频**：预览内播放 NAS 上的视频  
- **AI 智能搜索**：用语言描述搜索照片（依赖 NAS 侧能力）  
- **媒体分类 / 智能分类 / 人物**：按类型、智能归类、人脸聚合浏览（以 NAS 返回为准）  
- **预览页**：下载（图片多档、视频原文件）、系统分享、移入回收站删除、详情、查看原图  
- **上传**：从本机选图上传到飞牛相册  
- **设置**：保存连接与相册相关选项  

具体能力与数据依赖 NAS 版本与相册/AI 服务是否开通；详见主仓库 README。

## 使用注意与风险

- **非官方**：与飞牛官方无关，不保证与 NAS 升级后始终兼容。  
- **自行承担连接与账号安全**：地址、密码及网络环境由使用者自行负责。  
- **HTTPS / 证书**：可能出现证书提示，需自行判断是否信任。  
- **未签名包**：能否安装取决于设备与系统策略；勿安装来源不明的安装包。
