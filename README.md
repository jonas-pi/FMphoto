非官方 HarmonyOS 客户端，用于连接并访问 NAS 的常用能力（登录、网页入口、相册浏览与预览等）。

## 功能说明

- 仅测试过ipv6地址登录（HTTP/HTTPS，可手动切换）
- 支持账号密码登录与会话保持
- 支持首页 Web 内容加载与刷新
- 支持相册时间轴浏览、缩略图查看与大图预览
- 支持基础设置项保存（如地址、账号、部分会话配置）

## 使用方法

1. 安装并打开应用（首次启动需阅读并确认应用内免责声明）。
2. 在登录页填写服务器地址、用户名、密码。
3. 按需开启或关闭 HTTPS（未写端口时默认 HTTP:5666、HTTPS:5667）。
4. 点击登录，成功后进入主页或相册页面。

## 环境要求

- 一台可访问的飞牛 NAS（局域网或可达网络）
- 可用的 NAS 账号与密码
- HarmonyOS 设备（已允许网络权限）
