# FMphoto

第三方飞牛相册相关的原生鸿蒙应用**预编译包**说明仓库（根目录下的 `.hap` 为未签名安装包，仅供熟悉环境的用户自行安装测试）。源码与持续开发见：[FNOSforohos](https://github.com/jonas-pi/FNOSforohos)。

## 功能

- 连接飞牛 NAS，登录后使用网页入口与相册浏览
- 相册时间轴、缩略图、大图预览；可将手机照片上传到 NAS 相册
- 大图预览时可使用系统分享
- 设置中保存连接与相册相关选项

## 使用注意与风险

- **非官方**：与飞牛官方无关，不保证与 NAS 升级后始终兼容。
- **自行承担连接与账号安全**：地址、密码及网络环境由使用者自行负责。
- **HTTPS / 证书**：可能出现证书提示，需自行判断是否信任。
- **未签名包**：本仓库提供的安装包为未签名构建，能否安装取决于设备与系统策略；勿安装来源不明的安装包。
